# Calculatrice
Fonction simple d'une calculatrice en Javascript (addition-multiplication-soustraction-division)

Réalisation d'une mini-calculatrice avec 4 modes différents entre 2 nombres: 
  -  Addition
  -  Multiplication
  -  Soustraction
  -  Division

Le mode et les 2 nombres seront selectionnés par l'utilisateur lorsqu'il lancera le programme.
